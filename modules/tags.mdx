# Tags

Crea respuestas guardadas para reutilizar textos largos con un comando.

  <Card title="Gestionar" icon="list" href="#comandos" />

## Comandos

### <Icon icon="terminal" /> `;tag create <nombre> <contenido>`
- <Icon icon="chevron-right" /> **Que hace**: crea un tag.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras configurar o agregar algo nuevo.

### <Icon icon="terminal" /> `;tag delete <nombre>`
- <Icon icon="chevron-right" /> **Que hace**: elimina un tag.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando necesites eliminar o limpiar configuracion.

### <Icon icon="terminal" /> `;tag edit <nombre> <contenido>`
- <Icon icon="chevron-right" /> **Que hace**: edita un tag.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras actualizar el contenido del tag.

### <Icon icon="terminal" /> `;tag info <nombre>`
- <Icon icon="chevron-right" /> **Que hace**: muestra info (autor, usos).
- <Icon icon="chevron-right" /> **Cuando usar**: cuando necesites revisar el estado actual.

### <Icon icon="terminal" /> `;tag list`
- <Icon icon="chevron-right" /> **Que hace**: lista tags.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando necesites revisar el estado actual.

### <Icon icon="terminal" /> `;tag alias <tag> <alias>`
- <Icon icon="chevron-right" /> **Que hace**: crea un alias para un tag.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras crear un alias corto.

### <Icon icon="terminal" /> `;tag unalias <alias>`
- <Icon icon="chevron-right" /> **Que hace**: elimina un alias.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando necesites eliminar o limpiar configuracion.

### <Icon icon="terminal" /> `;tag search <texto>`
- <Icon icon="chevron-right" /> **Que hace**: busca tags por nombre.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras revisar una lista o historial.

### <Icon icon="terminal" /> `;tag raw <nombre>`
- <Icon icon="chevron-right" /> **Que hace**: muestra el contenido sin formato.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras ver el contenido sin formato.

### <Icon icon="terminal" /> `;tag claim <nombre>`
- <Icon icon="chevron-right" /> **Que hace**: reclama un tag abandonado.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras reclamar un tag abandonado.

### <Icon icon="terminal" /> `;tag transfer <nombre> @usuario`
- <Icon icon="chevron-right" /> **Que hace**: transfiere un tag.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras pasar la propiedad del tag.

## Uso rapido

- <Icon icon="terminal" /> `;tag <nombre>` para enviar el contenido del tag.
