# Snipe

Permite ver mensajes borrados o editados recientemente (segun permisos).

<CardGroup>
  <Card title="Ver" icon="eye" href="#comandos" />
  <Card title="Gestion" icon="list" href="#comandos" />
</CardGroup>

## Comandos

### <Icon icon="terminal" /> `;snipe`
- <Icon icon="chevron-right" /> **Que hace**: muestra el ultimo mensaje borrado del canal.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras ver el ultimo mensaje borrado.

### <Icon icon="terminal" /> `;editsnipe`
- <Icon icon="chevron-right" /> **Que hace**: muestra el ultimo mensaje editado del canal.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras ver el ultimo mensaje editado.

### <Icon icon="terminal" /> `;snipelist`
- <Icon icon="chevron-right" /> **Que hace**: muestra lista de snipes recientes.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando quieras revisar una lista o historial.

### <Icon icon="terminal" /> `;clearsnipe`
- <Icon icon="chevron-right" /> **Que hace**: limpia los snipes guardados del canal.
- <Icon icon="chevron-right" /> **Cuando usar**: cuando necesites eliminar o limpiar configuracion.

## Notas

- <Icon icon="chevron-right" /> Requiere permisos de moderacion para evitar abusos.
