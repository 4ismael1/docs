# Instalacion

Esta guia te lleva desde cero: invitar el bot, revisar permisos y dejarlo listo para configurarlo.

## Requisitos

- <Icon icon="chevron-right" /> Permisos para invitar bots en el servidor.
- <Icon icon="chevron-right" /> Acceso a la configuracion de roles.

## Invitar el bot

1. Usa el enlace de invitacion oficial.
2. En el selector de servidor, elige el servidor correcto.
3. Acepta permisos recomendados (puedes ajustar despues).
4. Verifica que el bot aparezca en la lista de miembros.

## Permisos recomendados

Para evitar errores, se recomienda otorgar:

- <Icon icon="chevron-right" /> Manage Guild
- <Icon icon="chevron-right" /> View Audit Log
- <Icon icon="chevron-right" /> Kick Members
- <Icon icon="chevron-right" /> Ban Members
- <Icon icon="chevron-right" /> Manage Roles
- <Icon icon="chevron-right" /> Manage Channels
- <Icon icon="chevron-right" /> Manage Webhooks
- <Icon icon="chevron-right" /> Manage Messages

## Verificar que esta online

- <Icon icon="chevron-right" /> El bot debe aparecer con estado activo.
- <Icon icon="chevron-right" /> Ejecuta `;ping` para comprobar latencia.
- <Icon icon="chevron-right" /> Ejecuta `;help` (si esta disponible) para ver comandos basicos.

## Paso siguiente

Una vez invitado, ve a **Permissions** para ajustar la jerarquia y evitar fallos de permisos.
