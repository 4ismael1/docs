# Licencias

El sistema de licencias activa el bot por servidor. Si tu servidor no tiene licencia activa, algunos comandos estaran bloqueados y el bot te mostrara un mensaje de activacion.

## Cuando necesitas una licencia

- <Icon icon="chevron-right" /> Si el bot esta en tu servidor pero responde con un aviso de licencia.
- <Icon icon="chevron-right" /> Si acabas de invitar el bot y aun no has activado el servidor.

## Comandos disponibles

### <Icon icon="terminal" /> `;license redeem <key>`

- <Icon icon="chevron-right" /> **Que hace**: canjea una licencia para este servidor.
- <Icon icon="chevron-right" /> **Quien puede usarlo**: administradores del servidor.
- <Icon icon="chevron-right" /> **Ejemplo**:

```text
;license redeem XXXX-XXXX-XXXX-XXXX
```

### <Icon icon="terminal" /> `;license status`

- <Icon icon="chevron-right" /> **Que hace**: muestra el estado de licencia del servidor.
- <Icon icon="chevron-right" /> **Cuando usarlo**: para confirmar si esta activa.
- <Icon icon="chevron-right" /> **Ejemplo**:

```text
;license status
```

## Que pasa si no hay licencia

- <Icon icon="chevron-right" /> El bot mostrara un mensaje de que el servidor no esta activado.
- <Icon icon="chevron-right" /> Se indicara el contacto de soporte para obtener una licencia.

## Problemas comunes

- <Icon icon="chevron-right" /> **Licencia invalida**: revisa que la key este bien escrita.
- <Icon icon="chevron-right" /> **Licencia ya usada**: la key esta ligada a otro servidor.
- <Icon icon="chevron-right" /> **Licencia revocada**: contacta soporte.
